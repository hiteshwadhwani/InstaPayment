
# InstaPayments


## Demo

https://insta-payment.vercel.app/

## Run Locally

Clone the project

```bash
git clone https://github.com/hiteshwadhwani/InstaPayment.git
```

Go to the project directory

```bash
cd InstaPayment
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm run dev
```
